ESLint + Prettier – שמירה על אחידות

← חזרה לדף הבית

למה ESLint ו־Prettier חשובים?

בפיתוח React, אחידות הקוד היא קריטית כדי לשמור על קריאות, תחזוקה ויכולת שיתוף פעולה בין מפתחים. כלים כמו ESLint ו־Prettier מסייעים לאכוף סגנון קוד אחיד ולמנוע שגיאות נפוצות בזמן הפיתוח.

  • מבטיח קריאות וסטנדרט אחיד לכל הקוד בפרויקט.
  • מונע באגים ושגיאות סינטקס מוקדמות.
  • משפר תהליכי Code Review על ידי הפחתת ויכוחים על סגנון קוד.
  • חוסך זמן ומאמץ בתחזוקה ארוכת טווח.

איך זה עובד

ESLint

מנתח את הקוד בזמן כתיבה או build ומזהה בעיות סינטקס, שגיאות לוגיקה ובעיות best practices. ניתן להגדיר חוקים מותאמים אישית (rules) לפי פרויקט או צוות.

דוגמא בסיסית (.eslintrc.json):

{
  "extends": ["eslint:recommended", "plugin:react/recommended"],
  "rules": {
    "no-unused-vars": "warn",
    "react/prop-types": "off",
    "no-console": "warn"
  },
  "env": {
    "browser": true,
    "es2021": true
  }
}

Prettier

עוסק בעיקר בעיצוב הקוד – spacing, line breaks, סוגי גרשיים, indentation. מסייע ליצור קוד אחיד ללא צורך בדיונים ידניים על סגנון.

דוגמא בסיסית להגדרה (.prettierrc או package.json):

{
  "singleQuote": true,
  "trailingComma": "es5",
  "semi": true,
  "tabWidth": 2,
  "printWidth": 100
}

שילוב ESLint + Prettier

  • ESLint מזהה בעיות לוגיקה וסגנון, Prettier מטפל בעיצוב אוטומטי.
  • שימוש ב־eslint-config-prettier מונע התנגשויות בין הכלים.

התקנה:

npm install --save-dev eslint prettier eslint-config-prettier eslint-plugin-prettier

הגדרה ב־.eslintrc.json:

{
  "extends": [
    "eslint:recommended",
    "plugin:react/recommended",
    "prettier"
  ],
  "plugins": ["prettier"],
  "rules": {
    "prettier/prettier": "error"
  }
}

טיפים למימוש יעיל

  • להגדיר את הכלים כבר בשלב תחילת הפרויקט.
  • לשלב עם IDE (VSCode) ל־Auto-Fix בזמן כתיבה.
  • לכלול בדיקות lint ב־CI/CD כדי לוודא אחידות גם בפרויקטים מרובי מפתחים.
  • להימנע מוויכוחים על סגנון קוד – הכלים מיישרים קו אוטומטית.

סיכום

שילוב ESLint ו־Prettier מבטיח קוד אחיד, קריא ותחזיק לאורך זמן. כלים אלו מקלים על מפתחים, משפרים תהליכי Code Review ומפחיתים באגים, במיוחד בפרויקטים גדולים ומורכבים.
